# Angular2-workshop

How to Start...

1. Make sure you have the nodejs installed.
2. Clone the repo:
    ```
    git clone https://github.com/nirmalkumar86/angular2-workshop.git
    
    cd angular2-workshop
    ```
3. Run the project.
    ```
      npm install
      npm start
      
    ```
4. It will automatically open http://localhost:3000 in browser. [make sure browser-sync installed, if its not happening].
